About the application:

1.It is game for two players, who take turns marking the spaces in a 3×3 grid. 
2.The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

Steps to run the Application :

1. Install node.js and gitbash.
2. In the gitbash use command "git clone 'https://github.com/navyabojja/tictactoe-react' " to clone the application.
3. install the packages by using command "npm install" in the application path.
4. Then start the application in the localhost by using command "npm start" .
